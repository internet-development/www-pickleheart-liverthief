# PICKLEHEART

Static Next.js 16 site for the **Pickleheart Liverthief** media art project. Three pages:

- `/` — landing page: hero quote, autoplay video, Zora NFT link.
- `/episodes/1` — Episode 1 video.
- `/attribution-non-commercial-share-alike` — full CC BY-NC-SA 4.0 license text.

Videos autoplay muted; clicking the overlay enables sound.

### Setup

```sh
npm install
npm run dev    # port 10002
npm run build
npm run start  # port 10002
```
